Community Updates
=================

If you'd like to stay up to date on the community and development of twosheds,
there are several options:

GitHub
------

The best way to track the development of twosheds is through
`the GitHub repo <https://github.com/Ceasar/twosheds>`_.

Twitter
-------

I often tweet about new features and releases of twosheds.

Follow `@Ceasar_Bautista <https://twitter.com/Ceasar_Bautista>`_ for updates.
